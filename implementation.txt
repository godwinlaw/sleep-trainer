     │ Sleep Training Tracker — Implementation Plan                                                   │
     │                                                                                                │
     │ Context                                                                                        │
     │                                                                                                │
     │ Building a greenfield sleep training tracker based on the "Twelve Hours in Twelve Weeks"       │
     │ methodology. The app tracks three pillars: daytime feeds (stretching intervals from 3h to 4h), │
     │  nighttime feeds (eliminating them), and naps (hitting duration targets). Currently only .env  │
     │ (Firebase config) and requirements.md exist — everything must be built from scratch.           │
     │                                                                                                │
     │ ---                                                                                            │
     │ Architecture                                                                                   │
